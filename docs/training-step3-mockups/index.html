Design Notes
  • All variants follow Civitai's dark theme design system (Mantine v7 + Tailwind)
  • Info icons are click-triggered popovers (not hover tooltips) for better mobile experience
  • Prompt textareas auto-grow as users type — no manual resize needed
  • Status badges indicate completion state ("3 prompts ready")
  • Advanced sections are collapsible to reduce visual clutter on first load
  • Core parameters (Steps, Checkpoints) remain always visible for quick scanning
  • All mockups use realistic placeholder content (Anima model, standard engine, etc.)
